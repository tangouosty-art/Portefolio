🐸 Exercice CSS — Flexbox Froggy

Comprendre justify-content

Cette page explique l’exercice de pratique CSS réalisé avec Flexbox Froggy. Le but est de déplacer les grenouilles horizontalement dans leur zone grâce à la propriété justify-content.

Lire l’explication Voir les captures
01

Objectif de l’exercice

Ce qu’il faut faire

Dans Flexbox Froggy, chaque grenouille doit être placée sur son nénuphar. Pour réussir, on écrit du CSS dans l’éditeur du jeu. Ici, l’exercice porte principalement sur l’alignement horizontal avec justify-content.

  • Déplacer les éléments vers la gauche, le centre ou la droite.
  • Répartir plusieurs éléments avec des espaces réguliers.
  • Comprendre l’axe principal d’un conteneur Flexbox.

La règle à retenir

justify-content agit sur l’axe principal. Par défaut, l’axe principal est horizontal parce que flex-direction vaut row. 

.pond {display: flex;
              justify-content: center;}
02

Explication simple

Pourquoi utiliser display: flex ?

Avant d’utiliser justify-content, il faut transformer le parent en conteneur flexible. Sans display: flex, la propriété ne fonctionne pas correctement sur les éléments enfants. 

.container {
  display: flex;
}

Comment choisir la bonne valeur ?

  1. Regarde où les grenouilles doivent aller.
  2. Si elles vont à droite : utilise flex-end.
  3. Si elles vont au centre : utilise center.
  4. Si elles doivent être espacées : utilise space-between, space-around ou space-evenly.
03

Les valeurs importantes

justify-content: flex-start;

Les éléments restent au début du conteneur, donc à gauche par défaut.

🐸🐸🐸

justify-content: flex-end;

Les éléments vont à la fin du conteneur, donc à droite par défaut.

🐸🐸🐸

justify-content: center;

Les éléments sont regroupés au centre du conteneur.

🐸🐸🐸

justify-content: space-between;

Le premier élément va au début, le dernier à la fin, et l’espace est au milieu.

🐸🐸🐸

justify-content: space-around;

Chaque élément reçoit de l’espace autour de lui.

🐸🐸🐸

justify-content: space-evenly;

Tous les espaces sont égaux : avant, entre et après les éléments.

🐸🐸🐸
04

Exercice à reproduire

Consigne

Crée un conteneur avec trois éléments, puis teste les différentes valeurs de justify-content. Observe comment les éléments se déplacent horizontalement. 

<div class="zone">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

Code CSS

.zone {
  display: flex;
  justify-content: space-between;
  align-items: center;
  height: 150px;
  padding: 20px;
}

.item {
  width: 50px;
  height: 50px;
}
